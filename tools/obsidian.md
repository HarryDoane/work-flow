# Obsidian

A local-first note-taking and personal knowledge base app. Notes are plain Markdown files stored on your own device (not locked into a proprietary format), with backlinks, a graph view, and a large plugin/theme ecosystem for customizing your workflow.

- Website: https://obsidian.md

## Install (macOS)

```sh
brew install --cask obsidian
```

## Pairs well with MarkItDown

Since Obsidian notes are just Markdown files, [MarkItDown](./markitdown.md) is a natural companion: convert PDFs, Word docs, spreadsheets, and other source material straight into Markdown and drop the output directly into an Obsidian vault.
